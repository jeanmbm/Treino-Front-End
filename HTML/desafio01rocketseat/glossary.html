Glossário

html
Delimita o início e o fim de um documento HTML. É o elemento raiz
head
Engloba os dados gerais (metadados) sobre o documento, como título, links, autor, dentre outros
title
Define o título do documento
meta
Fornece informações adicionais sobre o documento
body
Define o corpo do documento. Somente oque está dentro dessa tag será renderizado
p
Elemento que representa um parágrafo
a
Elemento que define uma hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.
div
Define um container geral que agrupa outros elementos
hr
Cria uma linha horizontal no documento
br
Destina-se a criar uma quebra de linha
