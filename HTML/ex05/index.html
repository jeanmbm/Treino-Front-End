No html, as tags de título/cabeçalhos são separadas por níveis, sendo elas:

Título nível 1 ( h1 )

Título nível 2 ( h2 )

Título nível 3 ( h3 )

Título nível 4 ( h4 )

Título nível 5 ( h5 )
Título nível 6 ( h6 )

O elemento h1 é utilizado como sendo o título principal do documento e os demais são os subtítulos, como vemos em jornais, revistas e outros.
